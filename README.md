# QCG IITR Recruitment Portal

Recruitment website for Quantum Computing Group, IIT Roorkee. Applicants fill out a form, take a domain-specific quiz, and see their results.

## Tech

- React + Vite
- Tailwind CSS v4
- React Router

## Run Locally

```bash
npm install
npm run dev
```

## Pages

- `/` - Home (about, contact)
- `/form` - Application form
- `/quiz` - Quiz based on selected domain
- `/result` - Score and feedback
